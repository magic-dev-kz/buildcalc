# 🧱 BuildCalc

Instant construction calculator — tiles, paint, flooring, concrete & more.

**[Try it →](https://magic-dev-kz.github.io/buildcalc/)**

## Features
- Tile, paint, flooring & concrete calculators
- Built-in unit converter (m², ft², gallons, liters…)
- Shopping list — save materials and quantities
- Dark / light mode
- Works offline — take it to the job site

## Why BuildCalc?
Contractors, DIYers, and homeowners waste time juggling formulas and unit conversions. BuildCalc puts every common construction calculation in one pocket-sized tool, so you can estimate materials on-site in seconds — no signup, no ads, no spreadsheets.

## Tech
- Single HTML file — nothing to install
- Works offline (PWA)
- No accounts, no tracking
- Your data stays in your browser (localStorage)
- Free forever
